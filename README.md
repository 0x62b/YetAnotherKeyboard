# YetAnotherKeyboard
This is an 65% mechanical keyboard with hot-swappable MX style switches. The design is based on a nice!nano microcontroller and a nice!view display.
It also supports wireless operation with a standard lithium polymer battery. The nano also supports a USB connection.

## Repo structure
CAD files including the individual parts and full assembly can be found in `cad/`
The KiCad PCB design files can be found in `PCB/`
Production gerbers can be found in `PCB/production`
Firmware can be found in `firmware/`

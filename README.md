# YetAnotherKeyboard
This is an 65% mechanical keyboard with hot-swappable MX style switches. The design is based on a nice!nano microcontroller and a nice!view display.
It also supports wireless operation with a standard lithium polymer battery. The nano also supports a USB connection.

## Repo structure
- CAD files including the individual parts and full assembly can be found in `cad/`
- The KiCad PCB design files can be found in `PCB/`
- Production gerbers can be found in `PCB/production`
- Firmware can be found in `firmware/`

## BOM
Total cost: 51.66 USD\
Parts can all be sourced from AliExpress.

## PCB
Parameters:
* 2 layer
* Material: FR-4
* Either HASL (leaded is much cheaper)
* White solder mask

Dimensions: 98.21mm*329.75mm\
PCB cost: 17 USD\
Shipping cost: 12.83 USD\
Total cost: 29.83 USD

## Case
Material: PLA, PETG, anything is fine really\
Nozzle: 0.4mm/0.6mm/0.8mm\
The case is far too big to fit on a standard 256mm^3 bed. If you want to replicate this build (for some reason) you may need to split the case. However I have access to a Prusa XL so I will use that.

## Firmware
ZMK, firmware is located in a submodule, https://github.com/0x62b/YetAnotherKeyboard-config
